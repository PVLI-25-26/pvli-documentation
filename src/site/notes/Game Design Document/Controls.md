| Action       | Gamepad (XBox Layout) | Keyboard & Mouse |
| ------------ | --------------------- | ---------------- |
| **Aiming**   | R-stick               | Mouse            |
| **Movement** | L-Joystick            | Arrows / WASD    |
| **Shoot**    | RT                    | LMB              |
| **Ability**  | RB                    | LShift           |
| **Interact** | A                     | E                |
| **Pause**    | Menu button           | Escape           |
